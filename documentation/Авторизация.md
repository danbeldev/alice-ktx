## Авторизация

[Документация](https://yandex.ru/dev/dialogs/alice/doc/ru/auth/when-to-use)


```kotlin
message {
    authorization(onAlreadyAuthenticated = {
        response {
            text = "Already Authenticated"
        }
    }, onAuthorizationFailed = {
        response {
            text = "Authorization Failed"
        }
    })
}
```
